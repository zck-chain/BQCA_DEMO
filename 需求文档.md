# 📂 AI 智能网盘转换工具 —— 需求文档

本项目是一个专为业务人员、开发人员打造的**非结构化数据提炼与 BQCA 自动绑定工具**。它将传统的 GCS 上传、BigQuery 表关联、大模型提取、人工校验以及 BQCA 知识库绑定的复杂操作，封装成一套**“流式自证、动态配置与降本增效”**的实用工具。

---

## 一、 工具定位与核心价值

本工具专注于实现以下极简流式工作流：

```
 用户上传文件到 GCS 
         │
         ▼
 用户配置分析参数（支持本地缓存防丢，提供一键恢复预设）
         │
         ▼
 后端动态触发大模型【冷热增量解析】并热部署 DDL 视图 (预览状态)
         │
         ▼
 前端大屏展示提炼结果、置信度及证据链 (用户肉眼核对)
         │
         ▼
 用户进行字段微调纠错 ➔ 点击“确认并生成表绑定 BQCA”
         │
         ▼
 后端在 BigQuery 生成【100% 准确且带描述注释的物理表】，一键完成 BQCA 语义绑定
```

**工具核心价值**：
1. **防丢与快捷恢复模板**：允许用户在前端微调提示词、温度和 Token。所有微调数据自动在浏览器本地进行 LocalStorage 缓存防丢失，并提供“一键恢复老麦黄金预设”功能。
2. **大模型增量解析降本**：系统自动对比“新上传文件”与“已解析历史数据”，**仅对增量新文件运行大模型解析**。对历史冷数据直接读取数据库缓存，避开重复 API 消费，费用直降 90%。
3. **高精准 BQCA 对话绑定**：在最终一键建表时，后端动态将中文字段解释以 `SET OPTIONS` 的形式写入 BigQuery 表元数据描述中。BQCA 检索时通过描述能 100% 懂用户的人话，杜绝聊天幻觉。

## 二、 核心功能步骤

### 1. 文件直传网盘 (GCS Upload)
*   提供一个极简、高级感的拖拽上传区域。
*   用户上传 PDF、图片、音视频文件，后台通过 **GCS Signed URL** 签发安全临时证书，前端直接将文件上传到 GCS 存储桶的专属临时目录下。

### 2. 分析参数配置与本地缓存面板 (UI Config Panel)
在触发分析前，前端提供一个折叠式“AI 智脑高级配置面板”：
*   **温度控制 (`temperature`)**：滑动条（0.0 ~ 1.0），0.0 表示极致严谨（适合财务法务），数值越高表示大模型越灵活。
*   **Token 限制 (`max_output_tokens`)**：大模型最大返回字数（如 512, 1024, 2048），以便平衡调用费用。
*   **提示词模板配置 (`prompts`)**：用户可直接在网页上修改 **【合同专家】、【简历猎头】、【发票财务】、【通用文档】** 这四个路由分类下的提示词模板。
*   **缓存防丢机制**：网页内所有改动均静默保存至浏览器的 `localStorage` 中。若网页不慎刷新，修改痕迹自动恢复；同时提供“恢复系统预设”按钮，允许用户一键重置。

### 3. 大模型增量解析预览 (Incremental AI Preview)
文件上传并配置好参数后，用户点击分析：
*   **增量解析控制**：后端自动对比当前待解析文件列表，排除已经完成提取、确认历史文件的 `uri`。**确保仅对新增的热数据运行大模型**，其余数据瞬间读取数仓缓存，耗时和 API 成本极佳。
*   **动态热部署**：后端将温度、Token 限制以及**用户微调后的提示词**注入 DDL 模板，调用 BQ SDK 在 BigQuery 瞬间编译热部署两阶段提取视图，并将结果拉回给前端预览。

### 4. 数据预览与双屏纠错 (Audit & Preview Panel)
*   前端以高保真表格呈现所有解析出的字段。
*   用户将鼠标悬停在字段上，可以直接查看 AI 的提取原文依据。
*   提供**双屏核对侧边板**，用户可以直接在网页上修改、订正提取不准确的字段值。

### 5. 语义级建表与 BQCA 一键绑定 (Generate & Semantic Bind)
当用户确认数据无误后，点击 **“确认并通过：生成物理表并绑定 BQCA”** 按钮。
*   **后台自动化 DDL 语义注入**：
    1.  后台接收到核对后的完美 JSON 数据，立刻在 BigQuery 中创建标准的物理表（如 `t_verified_smart_drive`）。
    2.  后端自动通过 BigQuery 的 **`ALTER COLUMN ... SET OPTIONS(description=...)`** 语法，将对应用中文字段注释注入底层数仓中。
    3.  系统调用 BQCA API 将该表一键绑定至聊天机器人。由于注入了精准的中文字段语义描述，BQCA 翻译 SQL 的准确度将达到 99% 以上，彻底免除幻觉！
